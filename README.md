# SBC Crypto Website

This is a sleek and dynamic cryptocurrency website built using HTML, CSS, JavaScript, and GSAP (GreenSock Animation Platform). The website is designed to provide a smooth, interactive, and visually appealing user experience, with responsive design ensuring compatibility across various devices.

## Live Demo

Check out the live demo of this app [here](https://harsh-login-app.vercel.app/).

## Features

- Responsive design: Ensures optimal display on devices of all screen sizes.
- GSAP Animations: Smooth, high-performance animations for enhanced user experience.
- Dynamic Content: Real-time updates and interactive features (can be enhanced with actual crypto data API).
- User-friendly interface: Clean and intuitive UI for browsing crypto information.

## Tech Stack

- **HTML**: The standard markup language used to structure the website.
- **CSS**: Used for styling the website and making it responsive.
- **JavaScript**: For interactive elements and dynamic functionality.
- **GSAP**: A powerful JavaScript library for creating high-performance animations.
